#### var stripped = stripFormatting(val)

Removes leading whitespace from each line of input
- **val** (*string*) - The string to strip formatting from

<!-- VDOC END -->

```javascript
const stripFormatting = require('strip-formatting')
stripFormatting(`
<html>
  <head>
    <title>whatever</title>
  </head>
  ...
`)
```
